## ctrlFOOD

Sistema para la venta de comidas y bebidas.

- Registro de usuarios
- CRUD producto
- CRUD clientes
- Registro de ventas
- Lista de ventas realizadas
- Reportes economicos
- Reportes estadisticos

Orientado a restaurants, fondas, pollerias, pensiones, franquisias de comida rapida, etc.
