## ctrlFOOD

Sistema para la venta de comidas y bebidas.

- Registro de usuarios
- CRUD producto
- CRUD clientes
- Registro de ventas
- Lista de ventas realizadas
- Reportes económicos
- Reportes estadísticos

Orientado a restaurants, fondas, pollerias, pensiones, franquicias de comida rápida, etc.
